Prediction Application for Telcom Customer Churn

Based on a Kaggle Dataset describing 7043 customers of a telecommunication company. Aim is to build and deploy a model to predict whether a customer is at risk for leaving the company.

Application website where you can try out the finished model.

The application was engineered using the Sklearn machine learning framework in a Jupyter Notebook IDE for model development; Flask was used to build the API and it was deployed with Heroku. Check out the code in my GitHub repository
